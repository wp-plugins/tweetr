== Description ==

Pulls the last user selected amount of tweets from your account and displays them in a sidebar widget

== Installation ==

1. Upload `Tweetr.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Move the widget into your themes sidebar 

== Frequently Asked Questions ==

= How do I link Tweetr to my Twitter Account? =

In Tweetrs settings page just enter your Twitter username and Tweetr will do the rest.
